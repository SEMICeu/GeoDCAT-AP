spatial resolution

rdfs:comment

dcat:spatialResolutionInMeters

GeoDCAT-AP 2.0.0

Until GeoDCAT-AP version 1.0.1 [[GeoDCAT-AP-20160802]], property rdfs:comment was used with class dcat:Distribution to specify the spatial resolution of a Distribution, since a more specific property was not available in reference vocabularies.

[[VOCAB-DCAT-2]] has defined a specific property to specify spatial resolution as distance in metres, namely, dcat:spatialResolutionInMeters, which has been adopted by [[DCAT-AP-20191120]]. Therefore dcat:spatialResolutionInMeters replaces rdfs:comment starting from this version of GeoDCAT-AP, but only when spatial resolution is expressed as distance in metres.
