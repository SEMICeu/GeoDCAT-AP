dqv:Dimension

This property represents the dimensions a quality metric, certificate and annotation allow a measurement of.

In GeoDCAT-AP, this property is used in the definition of the instances of dqv:Metric corresponding to the different type of spatial resolution. It MUST always take as value dqv:precision.

rdfs:DataType

This property represents the expected data type for the Metric's observed value (e.g., xsd:boolean, xsd:double, etc.).

In GeoDCAT-AP, this property is used in the definition of the instances of dqv:Metric corresponding to the different type of spatial resolution.
