+Service

A system that provides one or more functions [[DCTERMS]].

In the geospatial domain, a service is defined as the set of operations which may be performed, by invoking a computer application, on the spatial data contained in spatial data sets or on the related metadata [[INSPIRE-GLOSSARY]].

dctype:Service

dcat:DataService

GeoDCAT-AP 2.0.0

Until GeoDCAT-AP version 1.0.1 [[GeoDCAT-AP-20160802]], class dctype:Service was used to type resources as Data Services, as a more specific class was missing in reference vocabularies.

[[VOCAB-DCAT-2]] has defined a specific class for this purpose, namely, dcat:DataService, which has been adopted in [[DCAT-AP-20191120]]. Therefore, dcat:DataService replaces dctype:Service starting from this version of GeoDCAT-AP.

It is worth noting that dcat:DataService is a subclass of dctype:Service.
