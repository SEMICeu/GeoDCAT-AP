+Dataset Series

A collection of datasets that are published separately, but share some characteristics that group them.

dcat:Dataset

dcat:DatasetSeries

GeoDCAT-AP 3.x.x

Until GeoDCAT-AP version 2.0.0 [[GeoDCAT-AP-20201223]], class dcat:Dataset was used to type resources as Dataset Series, as a more specific class was missing in reference vocabularies.

[[VOCAB-DCAT-3]] has defined a specific class for this purpose, namely, dcat:DatasetSeries, which has been adopted in [[DCAT-AP-3]]. Therefore, dcat:DatasetSeries replaces dcat:Dataset starting from this version of GeoDCAT-AP.

It is worth noting that dcat:DatasetSeries is a subclass of dcat:Dataset.
