Usage note

An entity (e.g., an individual or an organisation) that is associated with Catalogues, Catalogue Records, Data Services, or Datasets. If the Agent is an organisation, the use of the Organization Ontology [[VOCAB-ORG]] is recommended. See for a discussion on Agent roles.
