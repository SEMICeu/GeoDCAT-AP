dcat:hadRole

GeoDCAT-AP 2.0.0

Until GeoDCAT-AP version 1.0.1 [[GeoDCAT-AP-20160802]], property dct:type was used with class prov:Attribution to specify the role of an Agent, as a more specific property was missing in reference vocabularies.

[[VOCAB-DCAT-2]] has defined a specific property for this purpose, namely, dcat:hadRole, which has been adopted in [[DCAT-AP-20191120]]. Therefore, dcat:hadRole replaces dct:type starting from this version of GeoDCAT-AP.
